Meus projetos:

Connect Dengue

O projeto foi idealizado durante o período do ensino médio no IFSP durante as aulas da disciplina de Projeto Integrador no ano de 2022. A ideia de criar um site que reportasse e mapeasse casos de dengue surgiu durante um momento de epidemia de dengue na cidade. A partir disso, chegamos a conclusão de criar um site que oferecesse esses recursos junto com uma ferramenta de denúncia que ofercece ao usuário a possibilidade de denunciar focos da doença a fim de previni-lá, além auxiliar na disseminação de informações. Por outro, lado também obtamos por criar uma API em Java que pudesse realizar as mesmas funções para complementar o projeto. Em Setembro de 2023 apresentamos o Connect Dengue na 13° Bragantec

Pequenos Guerreiros

Pequenos Guerreiros foi um projeto desenvolvido em grupo em 2024 durante as aulas da disciplina de Extensão no IFSP. O principal objetivo foi desenvolver um site para uma ONG mineira conhecida como Pequenos Guerreiros. Para desenvolvermos esse site obtamos por escolher um template e adapta-ló de acordo as necessidades da associação. Assim, Por meio de seu desenvolvimento pude exercitar minhas habilidades com html, css, e javascript aplicando assim, os conhecimentos adquiridos na sala de aula.

Cantina prática

A ideia partiu da tarefa de desenvolver um trabalho conclusiva para a matéria de Banco de Dados em 2022, logo pensamos em desenvolver um software de banco de dados em My Sql que pudesse auxiliar cantinas escolares. O projeto foi realizado em grupo num período de um mês nele desenvolvemos um modelo de banco de dados em que a cantina poderia registrar seus produtos, clientes, fornecedores e funcionários.
